Savitzky-Golay filter

savitzkyGolay(data: Array<Number>, h: Number, options: [Object]): Array<Number>
Parameters
data (Array<Number>) Array of y data
h (Number) Difference between the x dots
options ([Object]) Options object
Name Description
options.windowSize [Number] (default 5) Amount of dots used to make the filtering evaluation
options.derivative [Number] (default 1) Grade of the derivative
options.polynomial [Number] (default 2) Grade of the polynomial function to use for calculation
options.pad [String] (default 'none') How to pad the array to handle borders. Can be one of:
  • 'none': No padding. The resulting array will be smaller than the original one.
  • 'pre': Pad the original array before applying the filter.
  • 'post': Pad the resulting array after applying the filter.
options.padValue [String] (default 'replicate') If pad is not none, determine how to fill the values, if the value don't match with the next strings, the new values are going to be filled with that value.The special strings are:
  • 'circular': Pad with circular repetition of elements within the dimension.
  • 'replicate': Pad by repeating border elements of array.
  • 'symmetric': Pad array with mirror reflections of itself.
Returns
Array<Number>: Filtered data
